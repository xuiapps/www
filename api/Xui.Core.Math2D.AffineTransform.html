Struct AffineTransform

Namespace
Xui.Core.Math2D
Assembly

Represents a 2D affine transformation matrix in the form:

| A C Tx |
| B D Ty |
| 0 0  1 |

Used for rotating, scaling, translating, and skewing 2D vectors or points.

Remarks

Transformation is applied as:

| y' | = | B D Ty | * | y |
| 1  |   | 0 0  1 |   | 1 |
